# Truth Table Builder
Simple Python program that helps evaluating boolean formula and building truth tables

## Usage

`python main.py < filename`

or

`python main.py filename`

## Operations:

* Negation

  ~A

* Conjunction

  A & B

* Disjunction

  A | B

* Implication

  A -> B

* Equality

  A <-> B


## Example

(A & B -> !B) & (C | !A)
